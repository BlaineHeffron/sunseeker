# Contributing

## Development Setup

```bash
npm ci
npm run dev
```

## Before Opening a PR

Run:

```bash
npm run validate
```

Expected scope for changes:

- keep accessibility and responsive behavior intact
- preserve `/sunseeker` base-path compatibility
- include or update tests for user-visible behavior changes

## Commit Guidance

- Use clear, descriptive commit messages.
- Keep PRs focused and small enough for review.
